# Bash Scripting Exercises

A collection of beginner-friendly bash scripting exercises demonstrating core shell programming concepts including variables, conditional statements, loops, user input handling, and function definitions.

## 📋 Overview

This repository contains five bash scripts that serve as practical exercises for learning bash scripting fundamentals. Each script showcases different programming concepts and can be run individually or all together using the `five.sh` master script.
